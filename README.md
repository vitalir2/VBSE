# Bible of Software Engineering
Explanation of software engineering fundamentals using parallels in different fields of science and life.

## RU
За объяснением что лежит в этом репозитории, смотрите страницу [Мотивация](motivation_ru.md).

### Структура
Репозиторий состоит из томов и страниц.

**Страница** является файлом в Markdown формате. В странице идёт описание одной конкретной идеи, её объяснение и пару примеров из разных областей науки и жизни с фокусом на программирование.

**Том** является папкой. Каждый том состоит из двух или более страниц и имеет предисловие в виде `README.md` о содержании тома.

Репозиторий написан на двух языках: английский и русский. Найти страницу на необходимом языке можно по суффиксу `en` или `ru` или по названию папки внутри тома с такими же названиями.

## EN
For the explanation behind the repository contents please check [Motivation](motivation_en.md).

### Structure
The repository consists of volumes and papers.

**Paper** is a Markdown file. In a paper you can find a single idea, its explanation and a few examples from different fields of science and life with more focus on software engineering.

**Volume** is a folder. Each volume contains two or more papers where one of them is foreword in `README.md` about the contents of the volume.

The repository is written on two main languages: English and Russian. You can find the specific language paper with suffix `en` or `ru` accordingly or in a separate folder with the same names.
