# Bible of Software Engineering
Explanation of software engineering fundamentals using parallels in different fields of science and life.

## RU
За объяснением что лежит в этом репозитории, смотрите `motivation_ru.md` в корне.

### Структура
Репозиторий состоит из страниц, глав и томов.

**Страница** является файлом в Markdown формате. В странице идёт описание одной конкретной идеи, её объяснение и пару примеров из разных областей науки и жизни с фокусом на программирование.

**Глава** является папкой внутри тома, объединяющей несколько страниц в рамках общей темы.

**Том** является папкой. Каждый том состоит из двух или более страниц и имеет предисловие в виде `README.md` о содержании тома. Том может иметь внутри себя главы для структуризации страниц внутри.

Репозиторий написан на **двух языках**: английский и русский. Найти страницу на необходимом языке можно по суффиксу `en` или `ru` или по названию главы с такими же суффиксами.

## EN
For the explanation behind the repository contents please check `motivation_en.md` in the root.

### Structure
The repository consists of papers, chapters and volumes.

**Paper** is a Markdown file. In a paper you can find a single idea, its explanation and a few examples from different fields of science and life with more focus on software engineering.

**Chapter** is a folder inside the volume. It consists of many papers around specific topic.

**Volume** is a folder. Each volume contains two or more papers where one of them is foreword in `README.md` about the contents of the volume.

The repository is written on **two languages**: English and Russian. You can find the specific language paper with suffix `en` or `ru` accordingly or in a chapter with the same suffixes.
