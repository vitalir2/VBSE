# Абстракция (Абстрактный объект)
## Описание
Каждая вещь или идея в нашем мире является объектом, и если уйти от конкретного объекта и представить объект, объединяющий характеристики всех схожих объектов, то это будет называться **абстрактным объектом**. Для простоты будем называть это **абстракцией**, как и название самого метода перехода от частного к общему. Будь это книжка, теорема или любовь — всё это можно назвать абстракцией, пока оно не является конкретным томом книги Льва Толстого или теоремой Пифагора.

**Абстракция всегда является чем-то нематериальным**, но у каждой абстракции могут быть **проекции** на реальную жизнь (конкретные **объекты** в реальности). Например, теорема Пифагора, с одной стороны, является идеей, абстракцией, которая описывает определённую связь между сторонами треугольника, а с другой стороны — может быть записано на математическом языке на чём-либо, превращая формулу в надпись на экране или листке бумаги.

## Примеры
Это понятие одно из главных в любой науке так как позволяет выявить решения задач для любой ситуации, подходящих под условия, а не только в конкретных случаях. Примерами абстракций будут:
- в математике — число, неравенство, график
- в химии — молекула, соединение, химическая реакция
- в психологии — мышление, поведение, эмоция
- в правоведении — государство, закон, судья

Для аналогов в программировании можно посмотреть на следующий фрагмент кода ниже:
```
index = 0
sum = 0
elements = [1, 2, 3, 4]
while index < elements.size {
  sum += elements(index)
  index += 1
}
```
Для читателя вряд ли составит проблемы понять, что код выше суммирует числа от 1 до 4 в переменную sum, в итоге дав значение 10. Данный фрагмент кода описан на несуществующем языке программирования, но имеет внутри себя абстракции, которые делают его понятными для людей, знающих один или несколько языков программирования.
В частности абстракциями будут:
```
index = 0 <- переменная, присваивание, число
sum = 0
elements = [1, 2, 3, 4] <- массив (коллекция, кортеж, ..)
while index < elements.size { <- цикл, условие, размер массива
  sum += elements(index) <- доступ к элементу массива, присваивание со сложением
  index += 1
}
```

Все эти понятия описывают абстрактно то, что происходит в этом коде.
Но это только один из возможных вариантов выделения абстракций в этом фрагменте — подробнее в странице [Уровни абстракции](layers_of_abstraction_ru.md).
